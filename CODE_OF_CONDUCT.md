# Code of Conduct

Garda Code Action contributors are expected to keep discussion respectful, constructive, and focused on improving the project.

## Expected behavior

- Be respectful of different experience levels and backgrounds.
- Give clear, actionable feedback.
- Keep technical disagreements focused on the code and design.
- Avoid harassment, insults, or personal attacks.

## Enforcement

Maintainers may moderate issues, pull requests, discussions, or other project spaces to keep the community safe and productive.
